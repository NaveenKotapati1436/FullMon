# FullstackMon Website

A responsive, professional static website for FullstackMon Private Limited.

## Files
- `index.html` — main website page
- `styles.css` — full responsive styling
- `script.js` — mobile menu and dynamic year
- `assets/fullstackmon-logo.png` — company logo used in header and footer
- `assets/fullstackmon-icon.png` — logo icon
- `assets/favicon.png` — website favicon

## How to use
Open `index.html` in any browser, or upload the folder to any static hosting provider such as cPanel, Netlify, Vercel, GitHub Pages, Cloudflare Pages, or an S3 bucket.

## Notes
- The contact form uses `mailto:contact@fullstackmon.com`. Replace it with your real email or backend endpoint if needed.
- Legal details are based on public company records and the details provided by the user.
- The site avoids claiming official Dynatrace partnership. Add partner logos only if you have permission.
